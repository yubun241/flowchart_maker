## flowchart_maker
ブラウザ上で動作する、直感的で高機能なフローチャート作成ツールです。

## 概要
このツールは、プログラミングのアルゴリズムや業務フローを素早く描画するために設計されています。

HTML/JSのみで構築されており、インストール不要で動作します。

## 主な機能
- **多彩な形状**: 処理（矩形）、開始/終了（角丸）、分岐（菱形）、繰り返し（台形/逆台形）に対応。
  
- **4点接続システム**: 各図形の上下左右にある接続ポイントから、自由に矢印を引くことができます。
  
- **テキスト編集**: 図形だけでなく、接続した「線」の上にもラベル（True/Falseなど）を配置可能。
  
- **スタイルカスタマイズ**:
  
  - 図形ごとの背景色変更。
    
  - 背景色なし（透明）や枠線なしの設定が可能。
    
- **操作性**:
  
  - `Ctrl + Z`: 操作を元に戻す（履歴管理）。
    
  - `Backspace / Delete`: 選択した図形を削除。
    
  - `Escape`: 選択の解除。
    
  - `右クリック`: 接続線の削除。
    
- **PDF出力**: 作成したフロー図をそのままPDFとして保存・印刷が可能。

## 使い方
1. 左側のパネルから図形の形を選び、テキストを入力して「配置する」をクリックします。
   
2. 図形にマウスを重ねると表示される青い丸をドラッグして、他の図形へ繋ぎます。
   
3. 図形を選択した状態で左パネルのチェックボックスを操作すると、透明化や枠線の非表示ができます。
   
4. 完成したら「PDF出力」ボタンで保存します。
